CasaChiara - API Dati Immobiliari

Dati immobiliari italiani per l'estensione CasaChiara.

Endpoint

GET /omi.json — Prezzi medi di zona (468 comuni, 20KB)
GET /costi.json — NUOVO! Costi comunali reali: TARI, IMU, spese condominiali (70+ comuni, 8KB)

Formato omi.json

{
  "milano": { "r": [4.2, 23.4], "s": [1200, 5500] },
  ...
}

r = affitto [min, max] in €/m²/mese
s = vendita [min, max] in €/m²

Formato costi.json

{
  "citta": {
    "milano": { "tari": 3.68, "imu": 1.00, "condo": 2.03, "regione": "lombardia" },
    ...
  },
  "regioni": {
    "lombardia": { "tari": 3.28, "imu": 0.80, "condo": 1.45 },
    ...
  },
  "default": { "tari": 4.25, "imu": 0.68, "condo": 1.15 }
}

tari = TARI annua per m² (€)
imu = IMU effettiva come % del valore di mercato (annua)
condo = spese condominiali medie mensili per m² (€)

Fonti

OMI: Agenzia delle Entrate - OMI. Elaborazione onData.
TARI: Osservatorio prezzi e tariffe Cittadinanzattiva 2025.
IMU: Dipartimento Finanze MEF / Delibere comunali 2025.
Condominio: Indagine Changes Unipol / ANACI 2025.
