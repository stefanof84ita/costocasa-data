CasaChiara - API Dati OMI

Dati immobiliari dall'Osservatorio Mercato Immobiliare (Agenzia delle Entrate).

468 comuni italiani con prezzi medi di affitto e vendita al m².

Endpoint

GET /omi.json — Dataset completo (20KB)

Formato

{
  "milano": { "r": [4.2, 23.4], "s": [1200, 5500] },
  ...
}

r = affitto [min, max] in €/m²/mese
s = vendita [min, max] in €/m²

Fonte

Agenzia delle Entrate - OMI. Elaborazione onData.
